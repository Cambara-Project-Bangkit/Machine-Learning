# Machine Learning for Cambara
The Cambara application uses the application of Machine Learning to classify Aksara Bali images, using the Convolutional Neural Network (CNN) model.

## Library
- Numpy
- Tensorflow
- Cv2
- Keras
- Os
- Matplotlib
- Shutil
- Sklearn
